# Jerico

Jerico is a lightweight Linux security scanning CLI tool designed to help users quickly inspect their system for basic security-related information.

It focuses on:
- Running processes
- Disk and filesystem checks
- Privilege-related checks

This tool is intended for learning, system inspection, and basic security awareness.

---

## Features
- Process scanning
- Disk inspection
- Privilege checks
- Simple command-line interface
